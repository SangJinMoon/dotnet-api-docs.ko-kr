|   |   |
|---|---|

|제안 해결 방법|이 문제에 대 한 수정 릴리스에서 사용할 수는 서비스의.NET Framework 4.5.2 (핫픽스 롤업 3026376) 또는.NET Framework 4.6로 업그레이드 하 여|
|범위|주요함|
|버전|4.5.2|
|형식|런타임|
